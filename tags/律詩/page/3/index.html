茗汁詞

學淺無他藝，唯長雕琢詞。 沉吟常入定，點卯每來遲。 久病多清興，離群出詭思。 方酣濡墨盡，茗汁代塗之。

詠志

生似離弦箭，宜將鵠志求。 才情師子建，韜略效留侯。 過隧應斬岸，逾河當折舟。 無教箭虛落，白髮泣春秋。

李恥杜愁

性俗全無墨在胸，冥搜拙句砌機峰。 詩成引得謫仙恥，筆落愁催工部容。 既與深情詩韻繫，何教秉賦櫟樗庸。 如吾堪作違心事，簡朽帛殘枯筆封。

灣仔商廈觀圬者

久困雕欄內，而無玉砌求。 無乾圬者汗，盡濕富人樓。 貴族連三代，貧門到白頭。 笙歌聽愈擾，喋喋不安休。

夜讀

無意尋春色，離群獨筆耕。 風幽星火動，葉蕩夜蟲鳴。 案上半尋闊，心田萬畝宏。 思潮如碧海，引棹到三更。
